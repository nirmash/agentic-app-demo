# adcgen — AI-Powered Form Generator

## Project Context

**What:** CLI tool that generates HTML data-entry forms using AI (GPT-4o) and Eleventy. Describe a form in natural language, preview in ASCII art, launch a styled GitHub-themed site.

**Stack:** Node.js, Express, Eleventy, Commander, Inquirer, Primer CSS, PostgreSQL (optional)
